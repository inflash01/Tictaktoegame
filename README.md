# Tic Tac Toe Game

A simple console-based Tic Tac Toe game written in Python.

## How to Play

1. Run the game using `python tictactoe.py`.
2. Players take turns entering moves (1-9) corresponding to the board positions:
   ```
   1 | 2 | 3
   ---------
   4 | 5 | 6
   ---------
   7 | 8 | 9
   ```
3. The first player to get three in a row (horizontally, vertically, or diagonally) wins, or it's a draw if the board fills up.

## Features

- Two-player game
- Win and draw detection
- Input validation

## Requirements

- Python 3.x
